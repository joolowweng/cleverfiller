# CleverFiller

> [!Warning]
> Still in Development

## Overview

A tampermonkey script that fills form fields, using AI to find the best match data for the fields.

## Installation

1. Install [Scriptcat Extension](https://docs.scriptcat.org/) for your browser.
2. Import CleverFiller script to the extension with this [link](https://raw.githubusercontent.com/joolowweng/cleverfiller/main/dist/cleverfiller.user.js).
3. Enable the script in the extension.
